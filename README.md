convert-string-bytes
==============

JavaScript component to convert to/from strings and byte arrays.


Install
-------

### Node.js/Browserify

    npm install --save convert-string


Usage
-----

### stringToBytes(str)

### bytesToString(bytes)

### UTF8.stringToBytes(str)

### UTF8.bytesToString(bytes)
